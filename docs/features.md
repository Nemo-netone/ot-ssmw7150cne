│  ├─ 活动管理员登录
│  ├─ 用户资料维护
│  └─ 活动管理员资料维护
├─ 活动运营
│  ├─ 活动分类
│  ├─ 活动信息
│  ├─ 活动报名
│  └─ 活动跟踪
├─ 消息触达
│  ├─ 通知信息
│  ├─ 公告资讯
│  └─ 资讯分类
├─ 内容互动
│  ├─ 活动评论

## 用户场景

| 场景 | 用户动作 | 系统响应 |
|---|---|---|
| 浏览平台 | 打开首页 | 查看项目定位、活动模块和演示入口 |
| 管理员登录 | 使用 `admin / admin` | 进入后台仪表盘和全部管理模块 |
| 普通用户登录 | 使用 `用户账号1 / 123456` | 进入用户视角，查看报名、通知和收藏数据 |
| 活动管理员登录 | 使用 `管理账号1 / 123456` | 维护活动信息、审核报名和跟踪活动状态 |
| 查看活动 | 打开活动信息模块 | 读取 Supabase 中的活动种子数据 |
| 审核报名 | 编辑报名信息状态和回复 | Worker 更新 `baomingxinxi` 表 |
| 发布通知 | 新增或编辑通知信息 | Worker 写入 `tongzhixinxi` 表 |

## 模块责任

| 模块 | 数据表 | 责任 |
|---|---|---|
| 用户管理 | `yonghu` | 普通用户账号、联系方式、地址 |
| 活动管理员 | `huodongguanliyuan` | 活动管理员账号、资料、联系方式 |
| 活动分类 | `huodongfenlei` | 活动分类和筛选基础数据 |
| 活动信息 | `huodongxinxi` | 活动名称、地点、时间、报名要求和描述 |
| 报名信息 | `baomingxinxi` | 用户报名记录、审核状态和审核回复 |
| 活动跟踪 | `huodonggenzong` | 活动状态、图片和执行进度 |
| 通知信息 | `tongzhixinxi` | 用户通知、封面、时间和正文 |
| 公告资讯 | `news` | 平台公告和资讯内容 |
| 收藏管理 | `storeup` | 用户收藏活动或资讯 |

## 调用链

```text
用户操作
  -> site/app.js
  -> fetch /api/<module>/<action>
  -> site/_worker.js
  -> public.ot_ssmw7150cne_rest()
  -> ot_ssmw7150cne.<table>
```

## 数据边界

- 线上演示数据只放在 Supabase 的 `ot_ssmw7150cne` schema。
- Worker 只允许访问白名单表，避免任意表名访问。
- 原 MySQL dump 保留在 `db/ssmw7150cne.sql`，作为项目来源资料，不直接用于线上运行。
- 公开演示账号和种子数据经过简化，适合展示，不承载真实活动报名。
